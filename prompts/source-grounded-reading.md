# Prompt: Source-Grounded Reading

Copy and adapt:

```text
Use only the sources I provide in this chat or workspace.

Task: [compare / summarize / extract definitions / identify disagreement].

For every important claim, give:
1. the source name
2. the relevant passage or location if available
3. whether the claim is directly supported or inferred
4. what remains uncertain

Do not add outside knowledge unless I explicitly ask for it. If evidence is missing, say so.
```
