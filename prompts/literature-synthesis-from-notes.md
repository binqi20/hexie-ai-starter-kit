# Prompt: Literature Synthesis From Notes

Use this prompt only after you have individual research notes with evidence anchors.

```text
You are helping me synthesize a small literature set from verified notes.

Research question:
[insert question]

Inputs:
I will provide several research notes. Each note should include source status and evidence anchors.

Task:
Create a literature synthesis map, not a final essay.

Rules:
1. Use only claims present in the provided notes.
2. Do not add new citations or claim that a paper says something unless the note supports it.
3. Mark claims as high, medium, or low confidence based on source support and number of sources.
4. Separate consensus, disagreement, gaps, and possible implications.
5. Identify where I need to return to the original source.
6. Do not overstate causal claims unless the evidence design supports causal inference.

Output:
1. One-paragraph synthesis.
2. Theme map:
   | Theme | Sources | Shared claim | Tension or boundary | Evidence anchors |
3. Claim-evidence matrix.
4. Gaps and next searches.
5. Claims that need manual verification.
6. Possible outline for a short literature review.
```
