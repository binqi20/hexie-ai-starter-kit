# Prompt: Literature Search and Triage

Copy and adapt:

```text
My topic is: [topic].
My initial research question is: [question].
My field or pathway is: [GIS / SHR / SAH / other].

Help me refine this into 2-3 searchable research questions.

Then create a literature triage plan with:
1. key search terms
2. inclusion criteria
3. exclusion criteria
4. a table for recording candidate papers
5. questions I must answer before citing a paper

Do not invent citations. If you mention a paper, mark it as unverified unless I provide the source.
```
