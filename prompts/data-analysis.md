# Prompt: Data Analysis

Copy and adapt:

```text
I have a non-sensitive dataset.

Before analysis, first check:
1. whether the data description is sufficient
2. whether any column looks sensitive or identifying
3. row count and column names
4. missing values
5. possible data-quality issues

Then produce:
1. descriptive statistics
2. one useful chart
3. three cautious observations
4. assumptions and limitations
5. what I should verify manually

Do not make causal claims unless the research design supports causality.
```
