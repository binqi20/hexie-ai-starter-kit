# Claim-Evidence Matrix

Use this before asking AI to draft or revise writing.

| Section | Claim | Evidence anchor | Source ID(s) | Confidence | Possible overclaim | Revision needed |

| Introduction |  |  |  | high / medium / low |  |  |

| Analysis |  |  |  | high / medium / low |  |  |
| Implication |  |  |  | high / medium / low |  |  |

## Confidence guide

- High: direct source support, checked original source, clear anchor.
- Medium: source support exists, but context or wording needs nuance.
- Low: early idea, indirect support, or not enough evidence.

Low-confidence claims are for exploration, not final submission.
