# Evidence Anchor Table

Use this table before writing from AI-assisted notes.

| Claim ID | Claim | Source ID | Anchor type | Anchor location | Evidence summary or short quote | Support status | Human checked | Action |
|---|---|---|---|---|---|---|---|---|
| C1 |  | S1 | page / section / table / figure / paragraph / quote |  |  | SUPPORTED / PARTIAL / UNSUPPORTED / CONTRADICTED / NOT_CHECKED | yes / no | keep / revise / remove |
| C2 |  | S2 | page / section / table / figure / paragraph / quote |  |  | SUPPORTED / PARTIAL / UNSUPPORTED / CONTRADICTED / NOT_CHECKED | yes / no | keep / revise / remove |

## Rules

- Keep only `SUPPORTED` claims for confident statements.
- Use `PARTIAL` only with narrower wording.
- Remove `UNSUPPORTED` and `CONTRADICTED` claims.
- Do not use `NOT_CHECKED` claims in final work.
