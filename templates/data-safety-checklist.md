# Data Safety Checklist

Before uploading data to any AI tool, check:

| Question | Yes/No | Action needed |
|---|---|---|
| Does the data include names, student IDs, employee IDs, patient IDs, emails, phone numbers, or addresses? |  |  |
| Does it include HR, health, financial, legal, or confidential business information? |  |  |
| Is the data covered by research ethics approval, consent limits, or institutional policy? |  |  |
| Do I have permission to upload it to this tool? |  |  |
| Can I use synthetic, public, or de-identified data instead? |  |  |
| Have I checked the tool's account type and data-use policy? |  |  |

## Default rule

If you are unsure whether the data is safe to upload, do not upload it.
