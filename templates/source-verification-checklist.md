# Source Verification Checklist

Use this before putting an AI-assisted claim into coursework, a proposal, or a presentation.

| Check | Yes/No | Notes |
|---|---|---|
| I opened the original source. |  |  |
| The citation is real. |  |  |
| The cited passage supports the claim. |  |  |
| I checked the method or data behind the claim. |  |  |
| I recorded any limitation or uncertainty. |  |  |
| I did not rely only on an AI summary. |  |  |

## Final question

Would I be comfortable explaining this source and claim to my instructor?
